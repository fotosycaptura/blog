Midiendo la Velocidad de una Red e Internet con iperf3


En el mundo de la administración de redes, medir la velocidad de una red local o la conexión a Internet es una tarea que se suele realizar para comprobar que el proveedor brinda el rendimiento contratado. Una de las herramientas más utilizadas para este propósito es iperf3, que permite evaluar el rendimiento de una red de manera rápida y precisa. En este artículo (Siempre quise decir esto XDD), veremos cómo usar esta herramienta para medir tanto la velocidad de una red LAN como la velocidad de conexión a Internet.
